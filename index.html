ESS IA Designer

Choose IA type first. Then choose a DV grouped by topic. The program recommends suitable IVs, builds an AI prompt, imports the AI response, and exports a PNG proposal card.

1. Choose IA type

This filters the DV list so students only see choices that fit the type of investigation.

2. Choose a DV by topic

The DV is the thing students measure. These are grouped by topic to make browsing easier.

3. Choose a recommended IV

IVs are scored by tag overlap with the DV and whether they fit the IA type.

4. Copy this prompt into an AI

The AI should respond in valid JSON only. The program will import that JSON in the next step.
